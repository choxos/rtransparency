# Identify mentions of project numbers.

Returns the index of mentions such as: "NIH Project no. AB123."

Returns the index of mentions such as: "Remerciements Cette étude a été
financée par l’Institut de Recherches Médicales et d’Etudes des Plantes
Médicinales (IMPM)."

## Usage

``` r
get_project_acknow(article)

get_project_acknow(article)
```

## Arguments

- article:

  A List with paragraphs of interest.

## Value

The index of the paragraph of interest.

The index of the paragraph of interest.
